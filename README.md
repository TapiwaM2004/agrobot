## 📌 About

**AgroBot Pro** is an intelligent agricultural assistant designed specifically for Zimbabwean farmers and agro-dealers. It provides real-time crop advice, disease diagnosis through photo analysis, and local market insights — all accessible via WhatsApp or a web dashboard.

---

## ✨ Features

- 🤖 **WhatsApp Bot** — Conversational AI assistant accessible on any phone
- 📸 **Photo Analysis** — Upload crop images to detect diseases and pests
- 🌦️ **Weather & Season Guidance** — Farming advice tailored to Zimbabwe's climate
- 📰 **Agricultural News Feed** — Latest updates relevant to local farmers
- 💳 **Zimbabwe Payments** — EcoCash & Paynow payment integration
- 🛠️ **Admin Panel** — Manage users, content, and analytics
- 🌐 **Web Dashboard** — Full-featured React website for extended access
